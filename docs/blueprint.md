# **App Name**:AMIS DE LA FAMILLE AU RWANDA

## Core Features:

- Homepage Replication: Accurately replicate the family theme homepage, including all sections, sliders, and content layout
- Service Pages Duplication: Replicate all service pages (e.g., family implants, family bonds) with identical layout and content.
- Appointment Booking Form: Implement a functional appointment booking form, mirroring the original site's form.
- Contact Page Parity: Ensure the contact page (including map, contact form, and address details) is replicated accurately.
- Blog Page Imitation: Duplicate the blog layout and individual post designs, dynamically displaying blog content.
- Image Gallery Replication: Replicate the image gallery, including layout and image display, using the images from the demo website.
- Patient Review Display: Implement a section to display patient reviews in the same format as the original site.

## Style Guidelines:

- Primary color: Deep Blue (#192A56), reflecting professionalism and trust.
- Background color: Light Gray (#F4F6F8), providing a clean, modern backdrop.
- Accent color: Sky Blue (#A9D0F5), used for interactive elements and highlights.
- Body font: 'PT Sans', a humanist sans-serif, for readability and modern look.
- Headline font: 'Space Grotesk', a proportional sans-serif, for a techy, clean title feel.
- Use clean, modern icons in blue and gray tones, consistent with the original site's style.
- Maintain the original site's grid-based layout for all pages, ensuring consistency and responsiveness.
- Incorporate subtle fade-in and transition animations for a smooth user experience.
